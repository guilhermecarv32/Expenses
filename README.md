# Expenses App - Controle de Gastos Pessoais 📱💸

Este é um projeto desenvolvido como parte do **Curso de Flutter e Dart da Cod3r**. O objetivo da aplicação é gerir despesas diárias, permitindo ao utilizador acompanhar os seus gastos de forma visual e organizada.

## 🚀 Funcionalidades

- **Registo de Transações:** Adiciona título, valor e data de cada despesa.
- **Gráfico Semanal:** Visualização dinâmica da percentagem de gastos em cada dia da última semana.
- **Gestão de Dados:** Listagem de despesas e opção para eliminar registos.
- **Interface Adaptável:** Design focado em usabilidade com temas personalizados.

## 🛠️ Desafios Técnicos Superados

Durante o desenvolvimento, foram aplicados conceitos fundamentais de Flutter, incluindo:
- **Gestão de Estado:** Manipulação de listas e atualização em tempo real da interface (StatefulWidgets).
- **Comunicação entre Widgets:** Passagem de funções e dados através de construtores.
- **Resolução de Conflitos de Ambiente:** Ajustes finos em versões do Gradle (8.x) e Kotlin (2.x) para garantir a compatibilidade com as ferramentas mais recentes.
- **Null Safety:** Implementação rigorosa para evitar erros de inicialização (`late`) e garantir a robustez do código.

## 📦 Tecnologias Utilizadas

- **Flutter** (Framework)
- **Dart** (Linguagem de Programação)
- **Intl** (Formatação de moedas e datas)
- **Material Design**
